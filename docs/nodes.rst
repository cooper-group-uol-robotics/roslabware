Fisher PP14102
==============

Node for the Fisher PP14102 Balance

Commands
---------

* ``0`` : Tares balance
* ``1`` : Turns balance on
* ``2`` : Turns balance off
* ``3`` : Gets the ``stable_mass`` in mg
* ``4`` : Gets the ``mass`` in mg

IKA RET Control VISC hotplate
==============================

Node for the IKA RET Visc hotpalte

Commands
---------

* ``0`` : Start heating
* ``1`` : Stop heating
* ``2`` : Start stirring
* ``3`` : Stop stirring
* ``4`` : Set stir speed
* ``5`` : Set temperature
* ``6`` : Set speed and start stirring
* ``7`` : Set temperature and start heating
* ``8`` : Stop everything

Mettler Toledo Quantos QB1 dispenser
=====================================

Node for the Mettler toledo Quantos Solid dispenser

Commands
---------

* ``0`` : Start dosing
* ``1`` : Stop dosing
* ``2`` : Get door position
* ``3`` : Get sampler position
* ``4`` : Get head data
* ``5`` : Get sample data
* ``6`` : Move head pin
* ``7`` : Open door
* ``8`` : Close door
* ``9`` : Set sample position
* ``10`` : Set tapper before dosing
* ``11`` : Set tapper during dosing
* ``12`` : Set tapper int
* ``13`` : Set tapper duration
* ``14`` : Set target mass
* ``15`` : Set tolerance
* ``16`` : Set tolerance mode
* ``17`` : Set sample ID
* ``18`` : Set pan off
* ``19`` : Set algorithm
* ``20`` : Set Anti-static
* ``21`` : Dispense solid
